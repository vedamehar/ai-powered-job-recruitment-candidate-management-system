# AI-Powered Job Recruitment and Candidate Management System

Salesforce-based AI recruitment platform designed to improve hiring speed, decision quality, and process governance using AI insights and automation.

## Project Overview

This project centralizes recruitment activities in Salesforce and supports:

- AI-based candidate prioritization
- Data-driven hiring and salary recommendation support
- Automated approvals and notifications
- Role-based access for recruiter and hiring manager workflows
- Monitoring-focused post-deployment operations

## Repository Structure

- `phase-1-requirements/`: Requirement study and foundational setup evidence
- `phase-2-salesforce-development/`: Core Salesforce build milestones
- `phase-3-ui-ux-development/`: Lightning app and user experience configuration
- `phase-4-data-migration-testing-security/`: Data quality, testing, and security activities
- `phase-5-deployment-documentation/`: Final documentation, monitoring, and support notes

## Phase Summary

- Phase 1: Requirement mapping and initial object/field planning
- Phase 2: Account setup, objects, tabs, fields, validation rules, approvals, emails, Agentforce AI
- Phase 3: Page layout optimization, dynamic forms, and user setup
- Phase 4: Duplicate/matching controls, security profile activities, and validation
- Phase 5: Documentation and operational support model
